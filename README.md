# 🌦️ Weather-Web-App

It's a Live weather forecasting web app here you can easily check weather of multiple cities

---

## 🚀 Features

### 🚀 Get real-time temperature and weather conditions

Simply check your current location temperature by allowing location permission.

### Search weather by city name

You can easily get weather of any city all over the world.

### View humidity, wind speed, and more

You can check real-feel temperature, wind speed, sunrise and sunset.

### Simple and responsive UI

This web app works on all screen sizes.

## 📦 Installation

### Step 1: Create Project Folder

Create a new folder in local machine and open it on code Editor (VS code)

### Step 2: Install Tailwind CSS

Open terminal And run the command

```
npm install tailwindcss @tailwindcss/cli
```

## 🌍 Usage

* Allow location permission to check the current weather of your location.
* Or search any city name to get weather information.
* View temperature, humidity, wind speed, sunrise and sunset.
