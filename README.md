# 🌦️ Weather-Web-App
It's a Live weather forecasting web app here you can easily check weather of multiple cities

<dl>🚀 Features
    <dt>🚀 Get real-time temperature and weather conditions</dt>
    <dd>Simply check your current location temperature by allowing location permission.</dd>

    <dt>Search weather by city name</dt>
    <dd>You can easily get weather of any city all over the world.</dd>

    <dt>View humidity, wind speed, and more</dt>
    <dd>You can check real-feel temperature, wind speed, sunrise and sunset.</dd>

    <dt>Simple and responsive UI</dt>
    <dd>This web app works on all screen sizes.</dd>
</dl>
